# BOT_VK – Бот для социальной сети ВКонтакте
###### Расскажет вам о погоде и еще о чем-нибудь :)

___

### ОПИСАНИЕ ПРОЕКТА

Добро пожаловать в VK Бот! 
Данный бот был написан в качестве применочного бота для лекции ЦЦК 2023. Бот умеет показывать прогноз погоды при помощи отправки запросов на wttr, а также имееет удобную модульность и практичность.

### РАЗМЕЩЕНИЕ

✅ Получите токен от сообщества ВКонтакте

✅ Клонируйте репозиторий проекта "Bot_VK" и перейдите в него

```
git clone https://github.com/SerezhaM/Bot_VK.git
.git
cd Bot_VK
```

✅ Создайте и активируйте виртуальное окружение

```
python3 -m venv venv
source venv/bin/activate
```

✅ Обновите pip и установите необходимые зависимости из файла requirements.txt

```
pip install --upgrade pip
pip install -r requirements.txt
```

✅ Создайте файл .env и заполните его

```
touch .env
nano .env
TOKEN='<your_secret_token>'
```

✅ Запустите исполнительный файл main.py

```
python3 main.py
```
